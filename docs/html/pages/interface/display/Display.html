Display¶

Name: Display CLI Module

Description: Handles the control of display of text in CLI

interface.display.Display.Change(amount)¶

Changes the count of printed lines by summing up

Parameters

amount (int) – Amount to sum up to the current count

interface.display.Display.ClearDisplay()¶

Clears the amount of tracked lines printed from STDOUT

interface.display.Display.ClearPreviousLine()¶

Deletes the last line from STDOUT

interface.display.Display.Initialize()¶

Initializes the CLI interface display (run once per application instance)

interface.display.Display.Print(message, e='\n')¶

Prints a line and tracks the amount

Parameters

  • message (str) – Message to print to STDOUT

  • e (str) – Ending string for the message

interface.display.Display.PrintCount()¶

Prints the count of printed count
